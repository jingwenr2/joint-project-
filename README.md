# joint-project-

TTPR @baruch × Cybersecurity Reach Foundation research project — analyzing real honeypot attack data and ransomware victim data to publish a research report.

## About

This repo hosts our team's work for the [TTPR Research Project](https://ttpr.cybersecurityreach.org/), a ~6-week guided research program run with the Cybersecurity Reach Foundation (501c3). We're working with two real-world datasets:

- **Honeypot attack logs** (`events`) — 35M+ records of real intrusion attempts captured from live decoy servers
- **Ransomware victim data** (`ransomlook_posts`) — a running history of victims named on ransomware gangs' public leak sites

See [`PLAN.md`](./PLAN_en.md) for our research question, timeline, and team roles (currently in progress — direction not finalized yet).
